# Часть I, Задача 5

Пусть $L \leq X$ - собственное замкнутое подпространство, $y \not\in L \implies \exists f \in X^*$:

$$\|f\| = \frac{1}{\rho(y, L)}, \quad f(y) = 1, \quad L \subseteq \text{Ker } f$$

Определим функционал $g$ на $L_0 = \text{Lin}\{y\} \oplus L$

$x \in L \implies g(x + ty) = t \implies g(x) = 0$ для $x \in L$, $g(y) = 1$

Найдем $\|g\|$:

$$\|g\| = \sup_{\substack{x \in L \\ t \in \mathbb{C}}} \frac{|g(x + ty)|}{\|x + ty\|} = \sup_{\substack{x \in L \\ t \in \mathbb{C}}} \frac{|t|}{\|x + ty\|} = \sup_{\substack{x \in L \\ t \in \mathbb{C}}} \frac{1}{\|y + \frac{x}{t}\|} = (\text{т.к. } L \text{ - лин. пр-во}) =$$

$$= \frac{1}{\inf\limits_{x \in L} \|x + y\|} = \frac{1}{\rho(y, L)}$$

Искомый функционал $f$ - продолжение $g$ на $X$ по теореме Хана-Банаха.

*(Note: Если взять $F = \rho(y, L) f$, то $\|F\| = 1$, $L \subseteq \text{Ker } F$ - существование такого функционала требовалось в доказательстве леммы в п.1)*
