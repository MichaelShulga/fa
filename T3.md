# Т3

**Лемма:** Пусть $\{\alpha_k\}$ - ограниченная последовательность, $x_k \xrightarrow{w} 0$. Тогда $\alpha_k x_k \xrightarrow{w} 0$

$\triangleright |\alpha_k| \leq C \ \forall k \in \mathbb{N} \implies \exists f \in X^*$ Тогда $|f(\alpha_k x_k)| = |\alpha_k| |f(x_k)| \leq C |f(x_k)| \to 0 \implies \forall f \in X^* \ f(\alpha_k x_k) \to 0 \implies \alpha_k x_k \xrightarrow{w} 0 \ \square$

Пусть $\{x_n\} \subseteq S_1(0) \quad x_n \xrightarrow{w} 0$

Пусть $|\tilde{x}| < 1 \implies 0 \in B_1(0) \cap B_1(-\tilde{x})$

Тогда рассмотрим функцию $(h \in S_1(0)) \ \varphi_h(t) = \|ht + \tilde{x}\|$

$\varphi_h(0) < 1$, $\varphi_h(t) \geq t\|h\| - \|\tilde{x}\| \implies \lim\limits_{t \to +\infty} \varphi_h(t) = +\infty$, $\varphi_h \in C(\mathbb{R}) \implies \exists \ \tilde{t}_h : \varphi_h(\tilde{t}) = \|h\tilde{t}_h + \tilde{x}\| = 1$

$S_1(0) \cup S_1(-\tilde{x}) \subseteq B_2(-\tilde{x}) \implies \tilde{t}_h < 2$

Тогда пусть $t_n = \tilde{t}_{x_n}$. Тогда $t_n x_n \xrightarrow{w} 0$ по лемме. $\{x_n, t_n\}_{k \in \mathbb{N}} \subseteq S_1(-\tilde{x})$

Тогда пусть $\tilde{x}_n = x_n t_n + \tilde{x}$. $\{\tilde{x}_n\} \subseteq S_1(0)$ и $\tilde{x}_n \xrightarrow{w} \tilde{x}$

Значит, $B_1[0] \subseteq [S_1(0)]_{w\text{-секв.}} \subseteq [S_1(0)]_w \underset{(\text{по T2})}{=} B_1[0]$. То есть $\boxed{[S_1(0)]_{w\text{-секв.}} = B_1[0]}$
