Пусть $X$ - конечномерное ЛНП, $\{e_i\}_{i=1}^n$ - базис в $X$.

Пусть $\pi_j(e_i) = \delta_{ij}: \left(\pi_j\left(\sum\limits_{i=1}^n \alpha_i e_i\right) = \alpha_j\right)$ Пусть $f$ - линейный функционал на $X$:

$x = \sum\limits_i \alpha_i e_i \implies f(x) = \sum \alpha_i f(e_i) \implies$

1. Все нормы на $X$ эквивалентны $\implies$ если $\|x\| = \sum\limits_i |\alpha_i|$ для $x = \sum\limits_i \alpha_i e_i$, то

$|f(x)| \leq \sum\limits_i |f(e_i)| |\alpha_i| \leq \|x\| \cdot \sum |f(e_i)| \implies f$ непрерывен в $X$ с нормой выше $\implies$ в $X$ с любой другой нормой тоже

2. $\pi_i$ - ЛНЗ, в самом деле если $\sum\limits_k \beta_k \pi_k = 0 \implies \forall j \ \left(\sum\limits_k \beta_k \pi_k\right)(e_j) = \beta_j = 0 \implies \forall j \ \beta_j = 0$

$f = \sum\limits_i f(e_i) \pi_i \implies$

$\{\pi_i\}_{i=1}^n$ - базис в $X^* \implies \dim X = \dim X^*$. Тогда $\dim X^{**} = \dim (X^*)^* = \dim X^* = \dim X = n$

$\{e_i\}_{i=1}^n$ - ЛНЗ, $\{\varphi(e_i)\}_{i=1}^n$ - тоже ЛНЗ, их $n \implies \text{Lin } \{\varphi(e_i)\}_{i=1}^n = \varphi(X) = X^{**} \implies$

$X$ - рефлексивно
