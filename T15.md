# Т15

$A: L_2([0, 1]) \to c$

$$|Ax(k)| \leq \int_{-k}^k \frac{|x(t)|}{1 + t^2} \, dt \leq \int_{\mathbb{R}} \frac{|x(t)|}{1 + t^2} \, dt \leq C \|x\|_2, \ C = \sqrt{\int_{\mathbb{R}} \frac{1}{(1 + t^2)} \, dt}$$

Последнее неравенство из КБШ. Переходя к sup по $k$:

$\|Ax\|_{\infty} \leq C \|x\|_2 \implies A$ - ограниченный оператор.

$L_2([0, 1])$ рефлексивно $\implies$ по Т14 $A(B_1[0])$ замкнуто.

Для $y \in c \ \exists \ y(\infty) = \lim\limits_{k \to \infty} y(k)$.

$A(B_1[0])$ ограничено в силу ограниченности $A$.

$$|Ax(k) - Ax(\infty)| \leq \int_{-\infty}^{-k} \frac{|x(t)|}{1 + t^2} \, dt + \int_k^{+\infty} \frac{|x(t)|}{1 + t^2} \, dt \leq \left(2 \int_k^{+\infty} \frac{1}{(1 + t^2)^2}\right)^{\frac{1}{2}} \cdot \|x\|_2 = C_k$$

$$\forall x \in B_1^{L_2([0, 1])}[0] \implies |Ax(k) - Ax(\infty)| \leq C_k, \ C_k \to 0 \implies |Ax(k) - Ax(\infty)| \to 0 \text{ равномерно по шару} \implies A(B_1[0]) \text{ вполне ограничено.}$$

Найдем $A^*: \ell_1 \to L_2([0, 1])$.

$$y(1) Ax(\infty) + \sum_{k \in \mathbb{N}} y(k+1) Ax(k) = \int_{\mathbb{R}} \frac{y(1) x(t)}{1 + t^2} \, dt + \sum_{k \in \mathbb{N}} \int_{\mathbb{R}} \frac{y(k+1) I_{[-k, k]} x(t)}{1 + t^2} \, dt = (*) = \int_{\mathbb{R}} x(t) \cdot \frac{y(1) + \sum I_{[-k, k]} y(k+1)}{1 + t^2} \, dt$$

$(*)$: $\sum\limits_{k \in \mathbb{N}} y(k+1) I_{[-k, k]} \leq \|y\|_1 \implies$ интеграл и ряд можно переставить в силу Т. Лебега, взяв мажоранту $\frac{|x(t)| \cdot \|y\|_1}{1 + t^2}$

$\implies A^* y(t) = \frac{y(1) + \sum\limits_{k \in \mathbb{N}} y(k+1) I_{[-k, k]}(t)}{1 + t^2}$
