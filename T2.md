$B_1[0]$ $w$ - замкнут в силу теоремы Мазура

Пусть $|\tilde{x}| < 1$. Всякая $w$-окрестность $\tilde{x}$ содержит окрестность из базы вида $V = \left\{x : \max\limits_{1 \leq i \leq n} |f_i(x - \tilde{x})| < \varepsilon\right\}$. $\tilde{x} + \bigcap\limits_{i=1}^n \text{Ker } f_i \subseteq V$

Утв. $X$-бесконечномерно $\implies \forall n, f_1, \dots f_n \implies \bigcap\limits_{i=1}^n \text{Ker } f_i \neq \{0\}$

Тогда пусть $h \neq 0$, $h \in \bigcap\limits_{i=1}^n \text{Ker } f_i \implies \forall t \in \mathbb{R} \ \tilde{x} + ht \in V \implies$ Рассмотрим функцию

$\varphi: t \mapsto \|\tilde{x} + ht\| \quad \varphi \in C(\mathbb{R})$

$\varphi(0) < 1 \quad \varphi(t) \geq \|h\| |t| - \|\tilde{x}\| \implies \lim\limits_{t \to +\infty} \varphi(t) = +\infty \implies \exists \ \tilde{t} : \varphi(\tilde{t}) = 1$

$\implies \|\tilde{x} + \tilde{t}h\| = 1 \implies V \cap S_1(0) \neq \varnothing \implies$ это верно для всякой $w$-окрестности $\tilde{x}$ из базы $\implies$ для любой $w$-окрестности $\tilde{x} \implies \tilde{x} \in [S_1(0)]_w \implies B_1[0] \subseteq [S_1(0)]_w$. Обратное включение следует из того, что $S_1(0) \subseteq B_1[0]$, а $B_1[0]$ $w$-замкнут

Тогда $\boxed{[S_1(0)]_w = B_1[0]}$
