# Часть I, Задача 11

$$\varphi(f) = \int_a^b f g \, dx \quad f, g \in C([a, b]) \implies \begin{cases} f \in L_1([a, b]) \\ g \in L_{\infty}([a, b]) \end{cases}$$

Тогда в силу неравенства Гёльдера:

$$\|\varphi(f)\| \leq \|f\|_{\infty} \cdot \|g\|_1 \implies \text{разделив левую часть на } \|f\|_{\infty} \text{ и перейдя к sup по } f \neq 0:$$

$$\|\varphi\| \leq \|g\|_1$$

Рассмотрим функцию $s(x) = \text{sgn } g(x)$

$s \in L_1([a, b])$. Тогда $\forall \varepsilon > 0 \ \exists \ s_{\varepsilon} \in C([a, b]) : \|s - s_{\varepsilon}\|_1 < \varepsilon$

Рассмотрим функцию $\pi: \mathbb{C} \to \mathbb{C}$, $\pi(x) = \begin{cases} \frac{x}{|x|}, & |x| > 1 \\ x, & |x| \leq 1 \end{cases}$

Заметим, что $\pi$ - в точности проектор на замкнутый единичный шар.

В самом деле пусть $y \in B_1[0]$, $|x| > 1 \implies |x - y| \geq |x| - |y| \geq |x| - 1$. Заметим, что при $y = \frac{x}{|x|}$: $|x - \frac{x}{|x|}| = |x| \cdot |1 - \frac{1}{|x|}| = |x| - 1$ - неравенство обращается в равенство, а значит $|x - y|$ достигает минимума по $y \in B_1[0] \implies \pi(x)$ для $|x| > 1$ - проекция $x$ на $B_1[0]$

Для $x \in B_1[0]$ $\pi(x) = x$ и снова $\inf\limits_{y \in B_1[0]} |x - y|$ достигает минимума по $y \in B_1[0] \implies \pi$ - действительно проектор на $B_1[0]$

Тогда $\pi$ - липшицево отображение $\implies \tilde{s}_{\varepsilon} = \pi \circ s_{\varepsilon}$ - непрерывна как композиция непрерывных

Также $\forall x \in [a, b] \ |\tilde{s}_{\varepsilon}(x) - s(x)| = |\pi(s_{\varepsilon}(x)) - \pi(s(x))| \underset{s(x) \downarrow}{\leq} |s_{\varepsilon}(x) - s(x)| \implies \|\tilde{s}_{\varepsilon} - s\|_1 \leq \|s_{\varepsilon} - s\|_1 < \varepsilon$ и $\|\tilde{s}_{\varepsilon}\|_{\infty} \leq 1$ *(для чего и нужна композиция с $\pi$)*

Тогда наконец: $\varphi(\tilde{s}_{\varepsilon}) = \int_a^b (s - (s - \tilde{s}_{\varepsilon})) g \, dx \geq \|g\|_1 - \int_a^b |s - \tilde{s}_{\varepsilon}| |g| \, dx \geq \|g\|_1 - \varepsilon \|g\|_{\infty} \implies$

$\implies$ переходя к sup по $\varepsilon > 0$, получаем: $\|\varphi\| \geq \|g\|_1 \implies \boxed{\|\varphi\| = \|g\|_1}$

## Исследуем вопрос достижимости нормы:

Для начала заметим, что $\varphi(f) = e^{i\alpha} |\varphi(f)| = (\text{по линейности } \varphi) = \varphi(e^{-i\alpha} f) = \text{Re } \varphi(e^{i\alpha} f)$

$\implies \|\varphi\| = \|\text{Re } \varphi\| = \sup\limits_{\|f\|_{\infty} = 1} \text{Re } \varphi$. Заметим также, что если норма $\varphi$ достигается на некотором векторе, то существует такой вектор $h$, что $\varphi(h) = \text{Re } \varphi(h) = \|\varphi\|$ и $\|h\|_{\infty} = 1$. Очевидно, что существование такого $h$ - критерий достижимости нормы $\varphi$.

Пусть норма $\varphi$ достигается $\implies \exists h \in C([a, b])$:

$$\int_{[a, b]} h g \, dx = \int_{[a, b]} |g| \, dx$$

Пусть теперь $A \subseteq [a, b]$ - измеримое по Лебегу множество $\implies$

$$\int_A h g \, dx \leq \int_A |g| \, dx \text{ в силу неравенства Гёльдера.}$$

Аналогично: $\int_{[a, b] \setminus A} h g \, dx \leq \int_{[a, b] \setminus A} |g| \, dx \implies$ т.к. $\int_{[a, b]} h g \, dx = \int_{[a, b]} |g| \, dx$, неравенства обращаются в равенства.

Значит для всякого измеримого $A \subseteq [a, b] \implies \int_A h g \, dx = \int_A |g| \, dx \implies$

$\implies h(x)g(x) = |g(x)|$ для почти всех $x \in [a, b]$. Так как функции слева и справа непрерывны и равенства почти всюду следует тождественное равенство.

*(Иначе множество $\{x : h(x)g(x) \neq |g(x)|\}$ непусто и открыто, а значит имеет положительную меру)*

Имеем: $h(x)g(x) = |g(x)| \implies$ в точках множества $\{g \neq 0\}$ $h(x) = \text{sgn } g(x)$

Если же нашлась такая $h \in C([a, b])$, что $h(x) = \text{sgn } g(x)$ при $g(x) \neq 0$, то

$$\int_{[a, b]} h g \, dx = \int_{[a, b]} |g| \, dx \text{ и } \varphi \text{ достигает нормы.}$$

## Получаем критерий:

$\varphi$ достигает нормы, если и только если $\text{sgn } g$ можно непрерывно продолжить с $\{g \neq 0\}$ на весь отрезок $[a, b]$

---

## Примеры

### Задача 4

Рассмотрим $\varphi(f) = \int_a^b f(x - \frac{a+b}{2}) \, dx$

$\text{sgn}\left(x - \frac{a+b}{2}\right) = -1$ при $x < \frac{a+b}{2}$

$\text{sgn}\left(x - \frac{a+b}{2}\right) = 1$ при $x > \frac{a+b}{2}$

$\implies \varphi$ не достигает нормы, т.к. $\lim\limits_{x \to \frac{a+b}{2} - 0} \text{sgn}\left(x - \frac{a+b}{2}\right) \neq \lim\limits_{x \to \frac{a+b}{2} + 0} \text{sgn}\left(x - \frac{a+b}{2}\right)$

и значит нельзя найти такую $h \in C([a, b])$, что $h(x) = \text{sgn}\left(x - \frac{a+b}{2}\right)$ на $[a, \frac{a+b}{2}) \cup (\frac{a+b}{2}, b]$

### Задача 5

Пусть $x \not\in \text{Ker } f \implies f(x) \neq 0$

Тогда пусть $y \in X$.

Тогда $y = \frac{f(y)}{f(x)} \cdot x + \underbrace{\left(y - \frac{f(y)}{f(x)} x\right)}_{\in \text{Ker } f} \implies$

$y = tx + z$, где $t \in \mathbb{C}$, $z \in \text{Ker } f \implies$

$$\|f\| = \sup_{\substack{t \in \mathbb{C} \setminus \{0\} \\ z \in \text{Ker } f}} \frac{|f(tx + z)|}{\|tx + z\|} = \sup_{\substack{t \in \mathbb{C} \setminus \{0\} \\ z \in \text{Ker } f}} \frac{|f(x)|}{\|x + \frac{z}{t}\|} = \sup_{z \in \text{Ker } f} \frac{|f(x)|}{\|x + z\|} = \frac{|f(x)|}{\inf\limits_{z \in \text{Ker } f} \|x + z\|} = \frac{|f(x)|}{\rho(x, \text{Ker } f)}$$

Если нашелся элемент наилучшего приближения $\xi \in \text{Ker } f$, то $\rho(x, \text{Ker } f) = \|x - \xi\| \implies \|f\| = \sup\limits_{z \in \text{Ker } f} \frac{|f(x)|}{\|x + z\|} = \frac{|f(x)|}{\|x - \xi\|} = \frac{|f(x - \xi)|}{\|x - \xi\|}$

На вектор $x - \xi$ $f$ достигает нормы.

Пусть $f$ достигает нормы на векторе $y$: $\|f\| = \frac{|f(y)|}{\|y\|}$

$y = t_y x + z_y$, $t \neq 0$ (т.к. иначе $f(y) = 0$, а $\|f\| > 0$ т.к. $f \neq 0$) $\implies$

$$\|f\| = \frac{|f(t_y x + z_y)|}{\|t_y x + z_y\|} = \frac{|f(x)|}{\|x + \frac{z_y}{t_y}\|} = \frac{|f(x)|}{\rho(x, \text{Ker } f)} \implies \rho(x, \text{Ker } f) = \|x + \frac{z_y}{t_y}\| \implies -\frac{z_y}{t_y} \text{ - элемент наилучшего приближения в Ker } f \text{ для } x$$
