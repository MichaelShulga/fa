Пусть $\varphi$ - продолжение функционала предела с $c$ на $\ell_{\infty}$ с сохранением нормы

Тогда $\varphi(x) = \sum\limits_{k \in \mathbb{N}} \frac{1}{2^k} x(k) - \varphi(x)$. $\|\varphi\| = 1$, $\sum \frac{1}{2^k} = 1 \implies \|\varphi\| \leq 2$. Пусть $x_n(k) = \begin{cases} 1, & k \leq n \\ -1, & k > n \end{cases} \quad \|x_n\| = 1$

Тогда $\varphi(x_n) = 2 - \sum\limits_{k > n} \frac{1}{2^k} \to 2$, $n \to \infty \implies \|\varphi\| = 2$

В задаче 11 из части I было показано, что если норма функционала $\varphi$ достигается, то существует $h : \varphi(h) = \text{Re } \varphi(h) = \|\varphi\|$.

Тогда $\text{Re } \varphi(h) = \sum\limits_{k \in \mathbb{N}} \frac{\text{Re } h(k)}{2^k} - \text{Re } \varphi(h) = 2 \cdot \sum\limits_{k \in \mathbb{N}} \frac{\text{Re } h(k)}{2^k} \leq 1$ и $\text{Re } \varphi(h) \geq -1 \ (\|\varphi\| = 1)$.

Значит $\sum\limits_{k \in \mathbb{N}} \frac{\text{Re } h(k)}{2^k} = 1$, что возможно лишь в случае $\text{Re } h(k) \equiv 1$, $|h(k)| \leq 1 \implies h(k) \equiv 1 \implies \varphi(h) = 1 \neq -1 \implies$ такого $h$ нет

Таким образом, $\varphi$ не достигает нормы
